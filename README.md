# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

# Project Overview
This project presents an interactive IPL (Indian Premier League) Analysis Dashboard built using Microsoft Power BI.  
It provides detailed insights into team performance, player statistics, and season-wise trends from 2008 to 2025.

The dashboard is designed to be dynamic and reusable, allowing users to easily update it for future IPL seasons by simply modifying the dataset.

---

# Features
- Season-wise analysis (2008–2025)
- Points table & team performance
- Orange Cap (Top Run Scorer) insights
- Purple Cap (Top Wicket Taker) insights
- Most Fours & Sixes analysis
- Venue & match statistics
- Fully interactive filters and slicers

---

# Tools & Technologies
- Microsoft Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Excel / CSV Dataset

#  How to Use
1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Load or replace the dataset with new IPL season data  
4. Click on "Refresh"  
5. Explore the dashboard using filters and visuals
